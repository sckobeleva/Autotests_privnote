Автотесты для сервиса по созданию записок, уничтожающихся после прочтения. Учебный проект.  

В файлe requirements.txt находится конфигурация окружения для запуска тестов.
В файле conftest.py находится фикстура, запускающая и завершающая работу дайвера.
В файле fixture/application.py объявлен класс Application, которому принадлежат вспомогательные методы.
В файле test/tests.py находятся выполняемые тесты, которые можно запустить командой pytest -s -v tests.py
